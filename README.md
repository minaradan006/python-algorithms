# Python Algorithms

O colecție de algoritmi fundamentali de căutare, sortare și recursivitate, implementați în Python pentru studiul complexității și al eficienței codului.

## Algoritmi Implementați

### Recursivitate (Recursion)
*   **Fibonacci:** Calculul recursiv al șirului Fibonacci, ilustrând conceptele de apeluri în cascadă și optimizare structurală.
*   **Tower of Hanoi:** Rezolvarea clasică a problemei Turnurilor
  din Hanoi folosind divide et impera pentru mutarea eficientă a discurilor.

### Căutare (Searching)
*   **Binary Search:** Căutare binară în vectori sortați, cu o complexitate
  de timp optimă de O(log n).
*   **Bisection Method:** Metodă numerică de înjumătățire a intervalului
  pentru aproximarea rădăcinilor.
*   **Linear Search:** Căutare secvențială clasică, utilizată pentru
    verificarea elementelor în structuri de date neordonate (O(n)).

### Sortare (Sorting)
*   **Bubble Sort:** Algoritm simplu de sortare bazat pe compararea și
  interschimbarea repetată a elementelor adiacente.
*   **Merge Sort:** Algoritm stabil bazat pe paradigma divide et impera,
  garantând o complexitate de O(nlog n).
*   **Quicksort:** Sortare rapidă prin partiționare în funcție de un pivot,
  eficientă în practică datorită utilizării reduse a memoriei adiționale.
*   **Selection Sort:** Sortare prin selecție, ce funcționează prin găsirea
  repetată a elementului minim din partea nesortată.
